## Valentine Invite Site

Romantic single-page Next.js (App Router) site with playful Yes/No, floating hearts, countdown to February 13 at 9:00 PM, and formal instructions.

### Running locally
1. Install Node.js 18+.
2. Install deps: `npm install`
3. Start dev server: `npm run dev`
4. Open `http://localhost:3000`

### Deploying to Vercel
1. Install Vercel CLI: `npm i -g vercel`
2. Run `vercel` in the project root and follow prompts.
3. For updates, run `vercel --prod` or push to a connected Git repo.

### Assets
- Background music placeholder: `public/bg-music.mp3`
- Poster placeholder: `public/suzume-movie.jpg`

Replace these with your own files as desired.***
